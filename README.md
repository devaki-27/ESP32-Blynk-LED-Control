# ESP32-Blynk-LED-Control
ESP32 IoT project using Blynk Cloud. LED can be controlled through mobile app and physical push button. Includes LED status monitoring and button press counter.


Features:
- Control LED from Blynk App
- Manual ON/OFF using Push Button
- LED Status Feedback
- Button Press Counter

Hardware:
- ESP32 DevKit V1
- LED
- Push Button

Pins:
- LED -> GPIO 2
- Button -> GPIO 19

Libraries:
- Blynk
- WiFi
